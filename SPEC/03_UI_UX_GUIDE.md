# 添心小助手：UI_UX_GUIDE (視覺規範)

## 👩💼 小秘書核心機制 (Mascot)
- **視覺一致性**：通知氣泡頭像必須與主視窗服裝 **100% 同步**。
- **每天換裝**：啟動時隨機更換服裝 (Default, Blizzard, Thunder 等)。
- **對話序列 (MDQ)**：
    - **P1 (緊急)**：LINE/FB 訊息，立即覆蓋。
    - **P2 (狀態)**：打卡/任務成功，**鎖定 10s**。
    - **P3 (巡航)**：隨機閒聊。

## 💬 提醒氣泡規範 (Toast Notification)
- **物理參數**：400x180px，圓角 18px，左下角 20px 邊距。
- **頭像定位**：80x110px，`top center / cover` 確保臉部不被裁切。
- **自動關閉**：18s 自動倒數消失 (帶 Progress Bar)。
- **互動冷卻**：點擊「⏰ 稍後」進入 **20 分鐘** 冷卻排程。

## 📅 提醒中心邏輯
- **顯示原則**：今日所有任務 (System/iCloud/Local) 永久列出，完成後標記 `DONE` 沉底。
- **iCloud 同步**：每 30 分鐘同步一次；**15:30 後自動顯示明日行程預覽 (📅)**。
- **打卡連動**：若偵測到當日已有打卡時間，自動標記打卡提醒為 `DONE`。

## 🏆 里程碑鼓勵 (Milestone)
- **🌱 起步 (3項)**：溫暖認可。
- **🔥 加速 (6項)**：熱血鼓勵。
- **⭐ 卓越 (9項)**：專業讚賞。
- **🏆 全壘打 (12項)**：MVP 表揚。
- **🎖️ 傳奇 (15項)**：突破極限。

## 🎨 前端渲染策略
- 採用 **模板字面值 (Template Literals)**。
- 樣式必須具備特異性 (Scoped CSS)，防止汙染主進程或其它視窗。
