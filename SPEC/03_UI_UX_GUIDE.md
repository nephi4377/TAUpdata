# 添心小助手：UI_UX_GUIDE (視覺規範)

## 👩💼 小秘書核心機制 (Mascot)
- **視覺一致性**：通知氣泡頭像必須與主視窗服裝 **100% 同步**。
- **每天換裝**：啟動時隨機更換服裝，維持新鮮感與親和力。
- **對話序列 (MDQ)**：
    - **P1 (緊急)**：通訊訊息或系統崩潰，立即覆蓋。
    - **P2 (狀態)**：打卡/任務成功，**鎖定 10s** 展示成功狀態。
    - **P3 (巡航)**：背景閒聊與工作鼓勵。

## 💬 提醒氣泡規範 (Toast Notification)
- **物理參數**：400x180px，圓角 18px，具備動態進度條。
- **互動機制**：點擊「稍後」進入 20 分鐘冷卻期；點擊內容直接跳轉至對應功能。

## 📅 提醒中心與導覽邏輯
- **顯示原則**：今日任務永久列出，待辦置頂，完成項沉底。
- **上下文導覽 (Contextual Nav)**：統計看版中的 ❌ 或 📅 狀態標籤支援點擊，可直接調起個人設定中心。

## ⚙️ 設定視窗規範 (New Tabs)
- **分頁化架構**：採用「👤 身分切換」與「📅 同步設定」雙標籤，確保不同職責的功能不混淆。
- **視覺風格**：延續玻璃擬態設計，點擊按鈕具備 0.98x 的微動效反饋。

## 🏆 里程碑鼓勵 (Milestone)
- **🌱 起步 (3項)**：溫暖認可。
- **🔥 加速 (6項)**：熱血鼓勵。
- **⭐ 卓越 (9項)**：專業讚賞。
- **🏆 全壘打 (12項)**：MVP 表揚。
- **🎖️ 傳奇 (15項)**：突破極限。

## 🎨 前端渲染策略
- **Scoped CSS**：防止模組間樣式互相污染。
- **Template Literals**：利用原生 JS 模板字面值實現高性能渲染。
